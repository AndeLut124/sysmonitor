# ⚡ System Monitor

Мониторинг системы в реальном времени с красивым веб-дашбордом.

## Возможности

- 📊 **CPU** — загрузка процессора, график за последнюю минуту
- 💾 **Память** — использование RAM с кольцевым индикатором
- 💿 **Диски** — все разделы с заполнением
- 🌐 **Сеть** — скорость отправки/получения в реальном времени
- 📋 **Процессы** — топ-10 по загрузке CPU

## Установка и запуск

```bash
git clone https://github.com/AndeLut124/sysmonitor.git
cd sysmonitor
pip install -r requirements.txt
python run.py
